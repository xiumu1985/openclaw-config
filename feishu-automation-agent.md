# 飞书GUI自动化代理

## 项目目标
创建一个专门的GUI自动化代理来精确控制飞书应用，确保能正确发送中文内容给指定联系人。

## 功能要求
1. 启动和控制飞书应用程序
2. 精确识别GUI元素（搜索框、联系人列表、消息输入框等）
3. 正确处理中文输入和显示
4. 自动查找指定联系人
5. 发送中文消息给指定联系人
6. 错误处理和状态反馈

## 技术实现方案
- 使用Windows GUI自动化技术（如pyautogui、pygetwindow）
- 采用图像识别辅助定位GUI元素
- 中文输入法兼容性处理
- 消息发送确认机制

## 安全考虑
- 仅在授权的飞书应用窗口上执行操作
- 避免误操作其他应用
- 提供用户确认步骤
